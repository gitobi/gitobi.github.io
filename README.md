#James

Our Corporate website project. http://www.gitobi.com

## Requirements

* Yeoman
	* http://yeoman.io
* Ruby 2.1.2

## Environment

* using `Amazon S3`.

## Serve on local

```
grunt serve
```

## Deployment 

### Staging

```
grunt deploy
```

### Production

```
grunt deploy:production
```
