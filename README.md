#James

Gitobi LLC Corporate website.

## Requirements

* Yeoman
	* http://yeoman.io
* Ruby 2.1.2

## Environment

* using `Amazon S3`.

## Serve on local

```
grunt serve
```

## Deployment 

```
grunt build
grunt s3
```
